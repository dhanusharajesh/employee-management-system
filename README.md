# Employee Management System

A full-stack Employee Management System developed using FastAPI and React.

## Tech Stack

### Backend

* FastAPI
* SQLAlchemy
* SQLite

### Frontend

* React
* Vite
* CSS

## Features

* Employee Login
* Add Employee
* Edit Employee Details
* Delete Employee
* Search Employees
* Employee Dashboard
* Statistics Cards
* REST API Integration

## Project Structure

Employee-Management-System

├── employee_api

└── employee_frontend

## Backend Setup

1. Navigate to employee_api
2. Install required packages
3. Run the FastAPI server using Uvicorn

## Frontend Setup

1. Navigate to employee_frontend
2. Install dependencies
3. Start the Vite development server
